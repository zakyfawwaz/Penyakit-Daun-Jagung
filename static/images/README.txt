INSTRUKSI:
==========

1. Simpan foto pemandangan ladang jagung yang Anda kirim dengan nama:
   "corn-field-background.jpg"

2. Pastikan file berada di folder ini (static/images/)

3. Format yang didukung: JPG, JPEG, PNG

4. Ukuran file disarankan tidak terlalu besar untuk performa yang optimal

Catatan: Jika nama file berbeda, ubah nama file menjadi "corn-field-background.jpg"
atau ubah nama file di templates/login.html dan templates/register.html
